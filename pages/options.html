YouTube Blocker Setup

You must create a free key from YouTube Data API so that the extension knows what videos to block using that key.

Setup (10 minutes)

  1. Go to Google's Cloud Platform (GCP). You should already have an account if you are signed in to a google account. If not, then create a free account.
  2. Note: You do not have to set any payment method at any time to use this extension.
    • Question: Is YouTube Data API free? Yes. Your requests will stop working after you reach the daily API request quota of 10,000 units per day.
  3. Go to Console on the top right. Then create a new project and name it something like "My Personal YouTube Blocker Project". You can also click on the drop-down on the top left and click "New Project" from there.
  4. Click the top left menu and go to 'APIs and Services'
  5. You should be on your dashboard now. Click 'Enable APIs and Services', search for Youtube Data API v3, click it, and enable it. You should see a checkmark with 'API Enabled'
    • If you don't see that, then click 'Manage' and enable it on the Youtube Data Dashboard
  6. Go back to 'APIs and Services'
  7. Click credentials in the left hand menu and click 'Create Credentials' on the page then click API key in the drop down
  8. You will see your API Key below with a number and a warning symbol. Click the Api Key you created e.g. API Key 1
  9. Very Important: Click 'Restrict Key' and select 'Youtube Data API v3'.
    This is to limit your key usage to the Youtube Data quota of 10k/day and prevent others from using your key
    • Your 'API Key 1' should now have a checkmark to its left on the 'Credentials' page
  10. Copy and paste your new personal API key in the input below to register it to the extension. It will be a long text key like this: AIzaSyAeebo7DlkB6YyCem51Lq9AOAmFG
  11. Set your API key then press the Activate button on the extension. Refresh the video if it doesn't work right away. Double check your key is correct if doesn't work after 5 refreshes.
  12. Note: You do NOT have to set any payment method. If you did, then you can remove it since YouTube Data is free
